CPU (Central Processing Unit)
proses

Tujuan Pembelajaran
  • Siswa mampu menjelaskan CPU sebagai perangkat keras komputer beserta fungsinya.

CPU atau Processor adalah perangkat keras yang berfungsi memproses berbagai macam perintah. CPU dapat diibaratkan sebagai otak komputer. CPU mengontrol keseluruhan jalannya sistem komputer baik melakukan perhitungan atau menjalankan tugas. Model 3D dari CPU dapat kita lihat pada Gambar 1 berikut.

CPU bekerja dengan kecepatan sangat tinggi. Semakin cepat CPU menjalankan perintah, maka akses akan semakin cepat. Pada cpu terdapat 3 unit utama untuk melakukan tugas berbeda yaitu:

  • Aritmatics Logical Unit (ALU): melakukan semua perhitungan aritmatika (matematika)
  • Control Unit (CU): mengatur lalu lintas data seperti input output
  • Memory Unit (MU): memproses perintah penyimpanan dengan kecepatan tinggi

Beberapa merk cpu diantaranya Intel, AMD, IBM, Apple, Cyrix VIA, dan IDT. Namun, yang populer saat ini adalah Intel dan AMD. Lihat Gambar 2 berikut.
